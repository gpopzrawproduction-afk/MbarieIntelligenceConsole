# Mbarie Intelligence Console Installation Instructions

## System Requirements
- Windows 10 version 1809 (build 17763) or later
- .NET 9.0 runtime (self-contained, included)
- 200MB free disk space

## Installation Steps (MSIX)

1. **Extract the Installer**
   - Unzip the provided package to a folder (if delivered as a zip).

2. **Install the Application**
   - Double-click `MbarieIntelligenceConsole.msix`.
   - If prompted, click **Install**.
   - If SmartScreen or Windows Defender warns, click **More info** > **Run anyway** (if trusted).

3. **First Run**
   - Launch from Start Menu: **Mbarie Intelligence Console**.
   - On first run, follow the on-screen configuration wizard.

## Troubleshooting
- If installation fails, ensure Windows is up to date.
- For certificate errors, contact IT for a signed installer or install the provided certificate.

## Uninstallation
- Go to **Settings > Apps > Installed apps**.
- Find **Mbarie Intelligence Console** and click **Uninstall**.
