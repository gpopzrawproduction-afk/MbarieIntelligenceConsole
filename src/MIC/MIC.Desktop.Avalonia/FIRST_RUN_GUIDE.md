# First-Run Configuration Guide

Welcome to Mbarie Intelligence Console!

## First Launch Checklist
1. **Start the Application**
   - Open from Start Menu: Mbarie Intelligence Console

2. **Initial Configuration Wizard**
   - Enter your organization details (Company Name, Admin Email)
   - Set up the initial admin account (username, password)
   - Configure database connection (if prompted)
   - Review and accept license agreement

3. **Test Connectivity**
   - Ensure the dashboard loads and no errors are shown
   - Check that you can send a test email (if email integration is enabled)

4. **Review Settings**
   - Go to Settings > System Settings
   - Adjust preferences as needed (theme, notifications, etc.)

## Tips
- For advanced configuration, see the full documentation or contact support.
- All settings can be changed later from the Settings menu.
